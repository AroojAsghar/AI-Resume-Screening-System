**AI Resume Screening System**



**Overview**

The AI Resume Screening System is a machine learning-based recruitment application that automates resume screening and candidate recommendation. It extracts text from resumes, predicts the most suitable job role, and recommends candidates based on resume-job similarity.



&#x20;**Features**

* &#x20;Upload resumes in PDF, DOCX, JPG, and PNG formats
* &#x20;Automatic text extraction using OCR and PDF parsing
* &#x20;Resume preprocessing using NLP techniques
* &#x20;Job role prediction using Machine Learning
* &#x20;Cosine similarity-based candidate recommendation
* &#x20;Interactive Streamlit web interface
